## APIs

### Adobe Analytics API
The Adobe Analytics 2.0 APIs provide programmatic access to data, reports, and administration features within Adobe Analytics. They allow you to perform almost any action available in the Analytics user interface, including reporting, segment management, calculated metrics, and component administration.

### Adobe Analytics Bulk Data Insertion API
The Bulk Data Insertion API (BDIA) lets you upload server-side call data in batches of compressed CSV files instead of using client-side libraries such as AppMeasurement. It is the recommended successor to the single-event Data Insertion API for high-volume server-side data collection.

**Human URL:** [https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/bulk-data-insertion/](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/bulk-data-insertion/)

### Adobe Analytics Livestream API
The Livestream API is a reporting feature in Adobe Analytics that allows clients to receive traffic data processed by Adobe Analytics in real time. Hits are streamed to the client on a hit-by-hit basis in the same order and rate that they are processed by Adobe servers.

**Human URL:** [https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/livestream/](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/livestream/)

### Adobe Analytics Data Repair API
The Data Repair API allows you to permanently delete or transform data that has already been ingested in Adobe Analytics. It provides endpoints for estimating repair scope, creating repair jobs, and monitoring job status.

**Human URL:** [https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/data-repair/](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/data-repair/)

### Adobe Analytics Data Insertion API
The Data Insertion API allows server-side data submission to Adobe Analytics one event at a time using HTTP GET or POST requests. Unlike the Bulk Data Insertion API which processes compressed CSV files in batches, this API handles individual hit submissions in real time.

**Human URL:** [https://developer.adobe.com/analytics-apis/docs/1.4/guides/data-insertion/](https://developer.adobe.com/analytics-apis/docs/1.4/guides/data-insertion/)

### Adobe Analytics 1.4 API
The Adobe Analytics 1.4 APIs provide programmatic access to reporting, classifications, data sources, and report suite configuration. This version is deprecated and scheduled for end-of-life on August 12, 2026.

**Human URL:** [https://developer.adobe.com/analytics-apis/docs/1.4/](https://developer.adobe.com/analytics-apis/docs/1.4/)

## Features

| Name | Description |
|------|-------------|
| Real-Time Analytics | Access and analyze data in real time as visitors interact with digital properties. |
| Custom Segmentation | Build custom segments to isolate and analyze specific visitor groups and behaviors. |
| Calculated Metrics | Create derived metrics combining existing metrics with mathematical formulas. |
| Report Suites | Organize and partition data collection across multiple sites and business units. |
| Annotations | Mark specific dates or ranges in reports with notes for contextual analysis. |
| Bulk Data Insertion | Upload server-side event data in compressed CSV batches for high-volume collection. |
| Data Repair | Permanently delete or transform previously ingested data for privacy compliance. |
| Livestream | Receive real-time streaming hit data as each event is processed by Adobe servers. |
| Dimension And Metric Exploration | Programmatically discover all available dimensions and metrics in report suites. |
| Component Tagging | Apply tags to segments, metrics, and other components for organization and discovery. |

## Use Cases

| Name | Description |
|------|-------------|

| Customer Journey Optimization | Analyze multi-touch customer journeys to identify drop-off points and optimize conversion paths. |

| Audience Discovery And Targeting | Discover high-value audience segments for personalized marketing and advertising. |
