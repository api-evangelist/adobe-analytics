# Adobe Analytics

> Adobe Analytics provides real-time analytics and detailed segmentation across all marketing channels. The Adobe Analytics 2.0 APIs give programmatic access to reporting, segments, calculated metrics, dimensions, metrics, date ranges, annotations, and component administration — almost any action available in the Analytics UI. Companion APIs cover high-volume server-side data collection, real-time hit streaming, and privacy-driven data repair. Authentication is Adobe IMS OAuth (bearer access token) plus an Adobe Developer Console API key (x-api-key).

## APIs

- [Adobe Analytics API (2.0)](https://developer.adobe.com/analytics-apis/docs/2.0/): Reporting, segments, calculated metrics, dimensions, metrics, report suites, annotations, date ranges, tags. Base URL https://analytics.adobe.io/api/{globalCompanyId}
- [Bulk Data Insertion API](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/bulk-data-insertion/): Upload server-side call data in compressed CSV batches. Base URL https://analytics-collection.adobe.io/aa/collect/v1
- [Livestream API](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/livestream/): Real-time hit-by-hit streaming of processed traffic. Base URL https://livestream.adobe.net/api/1/stream
- [Data Repair API](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/data-repair/): Permanently delete or transform previously ingested data for privacy compliance. Base URL https://analytics.adobe.io/api
- [Data Insertion API (1.4)](https://developer.adobe.com/analytics-apis/docs/1.4/guides/data-insertion/): Single-event server-side submission (excluded from the 1.4 EOL). Base URL https://sc.omtrdc.net/b/ss
- [Adobe Analytics 1.4 API (deprecated)](https://developer.adobe.com/analytics-apis/docs/1.4/): Legacy reporting/admin; end-of-life August 12, 2026.
